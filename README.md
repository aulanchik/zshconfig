# ZSH Alias Profile Installer

A small CLI tool for managing modular ZSH aliases using **profiles**.

Instead of keeping a massive `.zshrc`, aliases are organized into sections that can be installed selectively.

---

# Features

- Modular alias profiles
- Simple CLI interface
- Automatic `.zshrc` integration
- Safe backups
- Dry-run preview
- Profile removal
- Profile discovery
- Colored output
